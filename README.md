# Curso de Python — Estudos e Projetos

Repositório pessoal para armazenar exercícios, projetos e anotações do curso de Python. Vou adicionar novos códigos e pastas conforme avanço nos estudos.

## Sobre
- Conteúdo: exercícios práticos, pequenos projetos e desafios do curso.
- Objetivo: organizar o aprendizado e demonstrar evolução no GitHub.

## Estrutura (exemplo)
- `001_mimica/` — projeto inicial com `app.py`.
- Novas pastas serão adicionadas conforme os módulos do curso.

## Requisitos
- Python 3.10+ (recomendo usar 3.10 ou 3.11).
- Recomendo criar um ambiente virtual para cada projeto.

## Como executar (Windows)
1. Criar e ativar ambiente virtual:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1  # PowerShell
```

2. Instalar dependências (se existir `requirements.txt`):

```powershell
pip install -r requirements.txt
```

3. Executar um exemplo (ex.: `001_mimica`):

```powershell
python 001_mimica\app.py
```

## Boas práticas
